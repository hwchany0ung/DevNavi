## 5개 섹션 카피

### 1. HERO — 첫 화면

**Badge:** AI가 만드는 나만의 커리어 플랜

**H1:**
```
어떤 개발자가 될지
아직 모르는 게 당연합니다
```

**H1 강조:**
```
3가지만 답하면
오늘부터 할 일이 생깁니다
```

**Subtext:**
목표 직군 · 목표 기간 · 지금 수준.
이 세 가지로 AI가 주차별 학습 계획을 만들어 드립니다.
"뭘 공부해야 하지?"에서 "이번 주엔 이걸 하면 돼"로.

**Primary CTA:** 지금 바로 만들어보기 (무료)
**Secondary CTA:** 샘플 로드맵 보기

**Social proof strip:**
- "9개 직군 지원"
- "평균 45초 만에 생성"
- "무료로 시작"

---

### 2. PROBLEM — 문제 제기

**Section Label:** 이런 적 있으셨나요?

**H2:** 로드맵을 찾다가 오히려 더 막막해졌다면

**Pain points (카드 3개):**

```
📌 "유튜브 로드맵 영상 다 달라요"
백엔드 로드맵 검색하면 나오는 영상마다
추천하는 스택이 다 달라서 뭘 믿어야 할지 모르겠음.
결국 아무것도 안 함.
```

```
📌 "리스트는 있는데 시작을 못 하겠어요"
노션에 공부할 것들 정리는 했는데
"이걸 다 해야 해?" 싶어서 오늘도 유튜브만 봄.
```

```
📌 "6개월째 기초만 반복하고 있어요"
JavaScript 기초 강의를 세 번 들었는데
이제 뭘 해야 할지 여전히 모르겠음.
```

**Bridge text:**
문제는 의지가 아닙니다.
오늘 뭘 해야 할지 명확하지 않은 것입니다.

---

### 3. SERVICE — 서비스 소개

**Section Label:** DevNavi가 하는 일

**H2:** 막연한 목표를 오늘 할 일로 바꿔드립니다

**How it works (3단계):**

```
Step 1 — 3가지 답변
목표 직군 (예: 백엔드)
목표 기간 (예: 6개월)
지금 수준 (예: Python 기초 완료)
```

```
Step 2 — AI 분석
입력한 정보를 바탕으로
어떤 스킬을 어떤 순서로 쌓아야 하는지
AI가 커리어 경로를 설계합니다.
```

```
Step 3 — 주차별 플랜
"1주차: Git 기초 + GitHub 레포 만들기"
"2주차: Python 함수·클래스 복습 + 미니 프로젝트"
이번 주 할 일이 명확하게 나옵니다.
```

**Feature highlights:**
- ✅ 중간에 방향 바뀌어도 재설계 가능 (지금까지 한 것 유지)
- ✅ 완료 체크 + 진행률 추적으로 동기부여 유지
- ✅ 직군별 핵심 자격증·포트폴리오 프로젝트까지 포함
